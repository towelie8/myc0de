# Lernen des CSS-Box-Modells durch das Erstellen eines Rothko-Gemäldes

Vor kurzem habe ich den Kurs **"Learn the CSS Box Model by Building a Rothko Painting"** auf FreeCodeCamp abgeschlossen. Dieser Kurs war eine fantastische Gelegenheit, um ein grundlegendes Konzept im Web-Design – das CSS-Box-Modell – durch das Erstellen einer künstlerischen Darstellung im Stil von Mark Rothko zu erlernen und zu vertiefen.

## Was ist das CSS-Box-Modell?

Das CSS-Box-Modell ist ein fundamentales Konzept im Web-Design, das beschreibt, wie die Elemente auf einer Webseite dargestellt werden. Jedes HTML-Element kann als eine Box betrachtet werden, die aus vier Hauptkomponenten besteht:

1. **Content** (Inhalt): Der eigentliche Inhalt der Box, wie Text oder Bilder.
2. **Padding** (Innenabstand): Der Raum zwischen dem Inhalt der Box und dem Rand (Border).
3. **Border** (Rand): Die Umrandung, die das Padding und den Inhalt umschließt.
4. **Margin** (Außenabstand): Der Abstand zwischen der Border der Box und anderen Elementen auf der Seite.

Durch das Verständnis und die Kontrolle dieser vier Komponenten können Designer das Layout und die Ästhetik einer Webseite präzise steuern.
