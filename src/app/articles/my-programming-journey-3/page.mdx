# Flexbox hilft dir, deine Webseite für jede Bildschirmgröße optimal zu gestalten

Im Rahmen meiner Weiterbildung auf FreeCodeCamp habe ich den Kurs "Flexbox helps you design your webpage so that it looks good on any screen size" absolviert. In diesem Kurs lernt man, wie man mit Flexbox ein responsives Layout erstellt, das sich automatisch an verschiedene Bildschirmgrößen anpasst. Eine der Hauptaufgaben war es, eine responsive Foto-Galerie zu entwickeln, die sowohl auf großen Bildschirmen als auch auf mobilen Geräten gut aussieht.

## Flexbox: Ein leistungsfähiges Werkzeug für responsive Designs

Flexbox, kurz für "Flexible Box Layout Module", ist ein CSS-Modell, das speziell dafür entwickelt wurde, um flexible und effiziente Layouts zu erstellen, die sich dynamisch an den verfügbaren Platz anpassen. Das Besondere an Flexbox ist seine Fähigkeit, Elemente in einem Container gleichmäßig zu verteilen und Abstände zu optimieren, ohne dass komplexes CSS oder JavaScript erforderlich ist.

## Kursinhalte und Lernerfahrungen

Der Kurs auf FreeCodeCamp ist in 24 Schritte unterteilt, die alle erfolgreich abgeschlossen wurden. Hier sind einige der wichtigsten Lektionen und Techniken, die im Verlauf des Kurses vermittelt wurden:

1. **Grundlagen von Flexbox**: Der Kurs begann mit einer Einführung in die Grundkonzepte von Flexbox, wie dem Verständnis von "main axis" und "cross axis", die entscheidend sind für das Ausrichten von Elementen innerhalb eines Flex-Containers.

2. **Flex-Container und Flex-Items**: Wir lernten, wie man einen Flex-Container definiert und die enthaltenen Flex-Items anordnet. Hierbei wurden die grundlegenden Eigenschaften wie `display: flex`, `justify-content`, `align-items` und `flex-direction` behandelt.

3. **Erstellen einer Foto-Galerie**: Die praktische Anwendung des Gelernten bestand darin, eine Foto-Galerie zu erstellen, die sich je nach Bildschirmgröße anpasst. Dabei haben wir gelernt, wie man Flexbox verwendet, um die Bilder gleichmäßig zu verteilen und sicherzustellen, dass sie auch auf mobilen Geräten optimal angezeigt werden.

4. **Responsives Design mit Media Queries**: Zusätzlich zur Nutzung von Flexbox wurde der Einsatz von Media Queries eingeführt, um noch flexiblere Designs zu erstellen, die auf verschiedene Bildschirmgrößen reagieren.
